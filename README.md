# 🛡️ Scam Awareness Educational Form

A professional, interactive educational form designed to raise awareness about common scam patterns including romance scams, cryptocurrency fraud, and other financial fraud schemes.

## 📋 Overview

This project demonstrates modern web development practices while serving an educational purpose. It helps users understand and recognize common scam patterns through a structured questionnaire.

### ✨ Features

- 🎨 **Beautiful UI Design**: Professional gradient backgrounds and smooth animations
- 📱 **Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices
- ✅ **Form Validation**: Real-time input validation with helpful error messages
- 🔒 **No Data Storage**: All data remains local and educational only
- ♿ **Accessible**: WCAG compliant design standards
- ⚡ **Smooth Animations**: Engaging user experience with CSS animations
- 📋 **Comprehensive Questions**: Covers multiple fraud types and scenarios

## 🎯 Form Sections

1. **Personal Background Information**
   - Age range, profession, tech-savviness level

2. **How Did You Meet?**
   - Platform details, initial contact, relationship development

3. **Scammer Details**
   - Profile information, red flags (checkbox options)

4. **Financial Information**
   - Fraud type, platform, amount range

5. **Timeline & Current Status**
   - Duration of contact, current status, last contact date

6. **Additional Details**
   - Money requests frequency, recovery hopes, additional info

## 🛠️ Tech Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with gradients and animations
- **Vanilla JavaScript**: Form validation and interactivity
- **No Dependencies**: Pure frontend solution

## 🎨 Customization

### Colors
Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
    --success-color: #10b981;
    /* ... more colors ... */
}
```

### Add/Remove Questions
Edit `index.html` and add new form groups:

```html
<div class="form-group">
    <label for="field-name">Question Here?</label>
    <input type="text" id="field-name" name="field-name" required>
</div>
```

### Change Colors of Gradient
Modify the gradient in `.header` and `body` selectors in `styles.css`
